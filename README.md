# Student-Automation-System
Bu proje, üniversite veya okul yönetim süreçlerini dijitalleştirmek için geliştirilmiş, Python tabanlı ve MySQL veritabanı entegrasyonuna sahip bir masaüstü otomasyon yazılımıdır. Modern bir arayüz (CustomTkinter/Tkinter) üzerinden öğrenci kayıtları, ders notu girişleri ve akademik veri listeleme işlemlerini gerçekleştirmeyi sağlar.🚀 Öne Çıkan ÖzelliklerÖğrenci Kayıt Yönetimi: Yeni öğrencilerin sistemdeki bölümlerle ilişkilendirilerek kaydedilmesi.Dinamik Veri Çekme: Bölüm listelerinin ve öğrenci bilgilerinin doğrudan MySQL veritabanından anlık olarak güncellenmesi.Not Giriş Sistemi: Öğrenci numarası ve ders kodu eşleşmesi üzerinden vize/final notlarının güncellenmesi.Gelişmiş Raporlama: Treeview kullanılarak tüm öğrencilerin bölümleri, notları ve ortalamalarıyla birlikte tek bir ekranda listelenmesi.İlişkisel Veritabanı: JOIN sorguları ile optimize edilmiş veri sorgulama mimarisi.🛠️ Kullanılan TeknolojilerProgramlama Dili: Python 3.xArayüz Kitaplığı: Tkinter & ttk (Modern UI bileşenleri)Veritabanı: MySQL (Relational Database Management System)Konektör: mysql-connector-python📋 Kurulum ve ÇalıştırmaProjeyi yerel makinenizde çalıştırmak için aşağıdaki adımları izleyin:1. GereksinimlerSisteminizde XAMPP veya benzeri bir yerel sunucu ile Python kurulu olmalıdır.2. Veritabanı YapılandırmasıXAMPP üzerinden MySQL'i başlatın ve yzp adında bir veritabanı oluşturun. Ardından aşağıdaki SQL tablolarının mevcut olduğundan emin olun:OgrencilerBolumlerDerslerOgrenci_Ders_Kayit3. Kütüphane KurulumuGerekli Python kütüphanesini yükleyin:Bashpip install mysql-connector-python
4. Uygulamayı BaşlatmaTerminal veya komut satırı üzerinden projeyi çalıştırın:Bashpython "ogrenci otomasyonu.py"
